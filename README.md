### 🌟 개요
**MAGIC Strategic Architect**는 DaeJung KIM dreamjkdj@gmail.com)이 고안한 **'MAGIC 프레임워크'**를 클로드(Claude) 에이전트 환경(Co-Work 등)에서 시각적 대시보드로 구현한 독보적인 전략 수립 스킬입니다.

이 스킬은 단순히 정보를 시각화하는 것을 넘어, 모호한 아이디어를 냉철한 의사결정 기반의 **'실행 가능한 전략'**으로 강제 전환시킵니다. 피드백 지옥에서 탈출하고, 확신을 가지고 작업을 종료(Close)하고 싶다면 이 지휘소(War Room)를 가동하세요.

### 🎯 핵심 기능 (M.A.G.I.C Logic)
사용자의 입력을 받으면, 5단계 프로세스를 거쳐 인터랙티브 HTML 대시보드를 즉석에서 생성합니다.

1.  **[M] Make it Clear (명확화)**: 문서/콘텐츠의 기능을 한 문장으로 고정 (승인/설득/전환 등).
2.  **[A] Add Context (맥락 추가)**: 현실 제약 조건(예산, 마감, 타겟)을 누락 없이 박제.
3.  **[G] Give Examples (예시 제공)**: 따라야 할 Best Practice(Must)와 피해야 할 Worst Case(Avoid) 대조.
4.  **[I] Include Goals (목표 설정)**: "어느 정도면 충분한가?"에 대한 5가지 Yes/No 통과 기준 설정.
5.  **[C] Check & Iterate (검증 및 반복)**: '의견'을 배제하고 단 3가지 핵심 수정 사항만 제안하여 작업을 즉시 종료.

## 🇺🇸 English Description

### 🌟 Overview
**MAGIC Framework Strategic Architect (By DaeJung KIM dreamjkdj@gmail.com)** is a groundbreaking strategic planning skill that implements the **'MAGIC Framework'**, conceived by Alex, author of "AI Work Revolution," as a visual dashboard within the Claude Agent environment (such as Co-Work).

Beyond mere information visualization, this skill forces ambiguous ideas into a cold, decision-based, **'executable strategy'**. If you want to escape feedback hell and terminate tasks (Close) with confidence, activate this War Room.

### 🎯 Key Features (M.A.G.I.C Logic)
Upon receiving user input, it generates an interactive HTML dashboard on the spot through a 5-step process.

1.  **[M] Make it Clear**: Locks the function of the document/content into a single sentence (Approval/Persuasion/Conversion, etc.).
2.  **[A] Add Context**: Etches in realistic constraints (Budget, Deadline, Target) without omission.
3.  **[G] Give Examples**: Contrasts best practices (Must) and worst cases (Avoid) to follow.
4.  **[I] Include Goals**: Sets 5 Yes/No passing criteria for "How much is enough?".
5.  **[C] Check & Iterate**: Excludes 'opinions' and suggests only 3 key revision items to terminate the task immediately.

### 🚀 How to Use (Based on Co-Work App)
1.  In the Co-Work environment, click the **[Add Plugin]** or **[+]** button.
2.  In the GitHub integration popup, enter the information below and click **[Sync]**.
    * **Owner**: `dreamjkdj-svg`
    * **Repo**: `visualize-magic`
3.  Find the `magic-architect.md` file in the list and click **[Install]**.
4.  Activate it by typing **"Plan a lecture using MAGIC"** in the chat window.
